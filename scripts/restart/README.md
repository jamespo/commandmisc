# umax-restart

Restart a UMax Hosting virtual machine with python & selenium

## Requirements
- python 3
- selenium
- chromium & chromedriver

## Config

Create config file ~/.config/.umaxcreds

    [credentials]
    username: myname@myemailaccount.com
    password: P234234&&^*&%

    [server]
    domain: vps.mydomain.net


## Usage

umax-restart
